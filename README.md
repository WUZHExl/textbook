# textbook

### 一、 项目介绍
基于 SpringBoot+Vue+Mysql 前后端分离的教材管理系统

角色：管理员、用户

用户：可以查询教材信息

管理员：可以对首页，用户管理，教材信息管理，教材订单管理，入库管理、出库管理等模块进行操作

### 二、项目技术
后端： SpringBoot + Mybaits-Plus

前端：Vue + ElementUI + HTML + CSS + JS

### 三、 运行环境
开发语言：Java、JS

项目架构：B/S架构

开发工具：IDEA、VScode

JDK版本：1.8
